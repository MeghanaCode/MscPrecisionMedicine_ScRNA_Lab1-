# PBMC Lab 1 – Single-Cell RNA-seq Workflow (Seurat)

This repository contains a complete, reproducible version of the PBMC 3k scRNA-seq workflow I implemented in R using Seurat.  
This analysis follows the classical single-cell pipeline:

### Steps included
- Load PBMC 3k matrix  
- Quality control (mitochondrial %, gene count thresholding)  
- Normalization  
- Highly variable gene detection  
- Scaling  
- PCA  
- UMAP  
- Graph-based clustering  
- Marker gene identification (FindMarkers, FindAllMarkers)  
- Visualization of clusters and marker expression  

### Skills demonstrated
- Single-cell RNA-seq preprocessing  
- Reproducible workflow design  
- High-quality visualization  
- Seurat proficiency (QC → clustering → DE)

**PBMC 3k** from 10x Genomics
